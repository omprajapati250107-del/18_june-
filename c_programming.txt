variable : A name which can store a value. 

    e.g. bottle = water     

            here, water is my value and bottle is a variable name 

        lvalue = rvalue 

        lvalue means variablename and rvalue means value 

    use of variable in program. 

    step 1 : variable declaration 

        syntax : datatypename  variablename; 

        e.g.    int  score; 

    step 2 : variable defination or variable initlization 

        syntax : variablename = value;

    step 3 : use of variable 

        syntax :    printf("%placeholder",variablename)

        e.g.  : printf("%d",score);

variable rules : 

1) we can't use predefined keyword as a variable name 

    e.g.  int  int X 
          int float X 

          int  INT correct

        e.g. int subject score   X 

3) variable name can contain only A to Z , a to z , 0 to 9 digits and _ (underscore) only.

        e.g. int subject_score; 

4) we can't start variable name with digits 

        e.g.  int  01num   X
